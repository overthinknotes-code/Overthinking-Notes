浪漫关系中的幻象递减函数

一个关于爱情认知结构的理论框架

关键词：幻象程度、浪漫关系阶段、理想化偏误、渐近线、扰动事件
摘要

本文提出一个描述浪漫关系中幻象程度随阶段推进而变化的理论框架，并以复合函数模型呈现其基准轨迹。核心命题为：个体对恋爱对象的幻象成分整体呈非线性递减趋势，但存在一条正值渐近线 ε（维持关系运作所需的最低必要幻象量）；每个关系阶段的临界点伴随短暂的蜜月期幻象反弹，随后回落至更低的基准水平；暗恋是全局幻象峰值所在，因其从未遭遇现实检验；丧偶后期因记忆美化机制产生纪念性幻象二次上扬。本文在第三节提供完整的公式推导与参数定义。本框架为思辨性理论，暂不具备实证量化基础，立足于现有心理学与哲学文献，意在为后续研究提供结构性参照。

第一节

理论背景

人类对恋爱对象的认知，从来不是对现实的精确映射。心理学家 Sandra Murray 与 John Holmes 的研究发现，关系满意度最高的伴侣，对对方的正面特质存在系统性高估，且这种理想化倾向在长期婚姻中依然具有功能性价值。[1] 完全去幻象化的关系往往更脆弱，因为幻象承担着情感缓冲的角色。

在哲学层面，普鲁斯特将爱情描述为一种认识论的失败：恋人以为自己在了解对方，实则不断在追赶一个永远在移动的版本。[2] 这与康德的物自体（Ding an sich）理论形成呼应——我们永远只能接触现象层，而无法触及事物本身。[3] 由此可知，幻象不只是认知偏误，而是人类认识他者这一行为的结构性约束。

Frankfurt 从意志哲学角度将爱定义为一种「在意」（caring）：爱一个人，意味着对方的幸福成为你意志结构的一部分。[4] 在这个定义下，幻象是意志投注的方向标——你首先在意的，是你心中关于那个人的形象，而非那个人的全部真实。以上研究共同指向一个悬而未决的问题：幻象程度随关系推进如何变化，其轨迹是否具有可描述的规律性？本文尝试为此提供框架性回答。

第二节

基准函数的定义

本文所提出的基准函数，描述的是统计意义上的普遍趋势，而非任何具体个体的真实轨迹。个体差异（依附风格、过往经历、自我意识程度等）以缩放系数 k 处理，不纳入基准曲线。

函数变量定义
X 轴浪漫关系的阶段（连续化处理）：陌生人 → 暗恋 → 热恋 → 婚姻 → 丧偶 → 丧偶后期
Y 轴幻象程度 I(x)：数值越高，认知中幻象（投影、理想化、信息填补）所占比例越高；数值越低，认知越接近对方真实面貌。
ε正值渐近线：Y 轴理论下界，代表维持关系运作所需的最低必要幻象量。
k个体缩放系数：影响曲线振幅，不改变整体形态。
P扰动参数：出轨、异地恋等非线性事件引发的局部波动，独立处理，不纳入基准曲线。

整体曲线呈非线性递减趋势，但并非单调下降：每个阶段临界点伴随短暂的蜜月期反弹，随后在现实持续作用下回落至更低基准。此结构重复出现，形成整体向下的不对称锯齿形。暗恋阶段为全局峰值，且其内部只有极缓慢的自然下降（无蜜月期反弹），因为它从未遭遇现实的冲击。

图1．浪漫关系幻象程度基准函数（Illusion Degree Baseline Function）
X 轴为关系阶段，Y 轴为幻象程度。虚线 ε 为正值渐近线。暗恋为全局峰值；热恋与婚姻入口各有蜜月期反弹（峰值依次递减）；丧偶后期有纪念性幻象二次上扬。图示为 k=1 基准曲线。

第三节

公式推导与参数定义

3.1　完整公式

基准函数由五个组成项叠加而成：

I(x) = ε + k · [ D(x) + P(x) + H₁(x) + H₂(x) + B(x) + M(x) ]
其中 I(x) 为幻象程度，x 为关系阶段（连续化参数，0–13），ε 为正值渐近线，k 为个体缩放系数。

3.2　各项定义

D(x)　基线衰减项
D(x) = (A₀ − ε) · e−αx

描述幻象程度随关系推进而产生的整体指数衰减趋势。A₀ 为陌生人阶段的初始幻象基线，α 为衰减速率。衰减趋近于 ε 而非零，反映正值渐近线约束。

P(x)　暗恋全局峰项
P(x) = A · exp(−(x − μc)² / (2σc²))

标准高斯核，描述暗恋阶段的幻象峰值。A 为峰值振幅，μc 为峰值位置（暗恋中段），σc 为宽度参数。此项产生整个函数的全局最高点，因为暗恋是幻象从未遭遇现实检验的阶段。

H₁(x)、H₂(x)　蜜月期反弹项
Hᵢ(x) = hᵢ · exp(−(x − μᵢ)² / (2σᵢ²)) · exp(−λᵢ · max(0, x − μᵢ))

不对称峰函数：高斯核（描述蜜月期的快速上扬）乘以单边指数衰减项（描述进入新阶段后幻象被现实逐步替换的下降过程）。H₁ 对应热恋入口蜜月期，H₂ 对应婚姻入口蜜月期。约束条件：h₁ > h₂，且两者峰值均低于 P(x) 的全局峰。

B(x)　丧偶骤降项
B(x) = −b · exp(−(x − μb)² / (2σb²))

负值高斯核，描述丧偶事件引发的幻象骤降。此项使 I(x) 在丧偶阶段触及或接近渐近线 ε——丧偶是函数中少数几个逼近 ε 下界的时刻：现实以不可回避的方式介入，过往建构的幻象结构受到冲击。

M(x)　纪念性幻象项
M(x) = m · exp(−(x − μm)² / (2σm²)) · 𝟙(x ≥ xbereave)

正值高斯核，附加阶跃函数约束（仅在丧偶之后生效），描述丧偶后期记忆美化机制（Fading Affect Bias）引发的幻象二次上扬。此上扬性质与蜜月期不同：蜜月期幻象是期待性的，朝向未来；纪念性幻象是回顾性的，朝向一个已封存的过去，因此不再进一步下降，形成新的高原。

X 轴将离散的关系阶段连续化处理，以支持公式的数学表达：

3.5　约束条件摘要

理论要求以下大小关系成立（与参数具体数值无关的结构性约束）：

max[P(x)] > max[H₁(x)] > max[H₂(x)] > 0
暗恋全局峰 > 热恋蜜月峰 > 婚姻蜜月峰
I(x) > ε，对所有 x 成立（丧偶最低点除外，允许趋近 ε）
M(x) = 0，当 x < xbereave
纪念性幻象仅在丧偶之后生效
λ₁ > λ₂（热恋蜜月退潮速度快于婚姻蜜月）
第四节

各阶段心理机制论述

4.0　关系经验的累积效应：蜜月期峰值的递减规律

个体系数 k 最重要的具体体现之一，是关系经验的累积会系统性地压低每段新关系的蜜月期峰值。第一段浪漫关系的蜜月期，是整个人生函数中幻象浓度最高、持续时间最长的节点：神经系统第一次被完整激活，无防御机制，无参照系。此后每段关系的经历都在更新内部认知图式（schema revision）——你在幻象展开之前已预见其将收缩，那个「预见」本身就占据了幻象原本可以填满的空间。[10]

4.1　陌生人阶段

陌生人阶段的幻象程度并非为零——人类的模式识别系统会自动以既有的内在原型（荣格所称的 Anima/Animus）对潜在对象进行初步映射。[5] 曲线起点已在 ε 之上，但处于整个函数中相对偏低的区间——明显低于暗恋阶段。随着浪漫意图的萌生，曲线末尾呈向上趋势，标志着即将进入幻象浓度最高的暗恋区间。

4.2　暗恋阶段（全局峰值）

暗恋是整个函数中幻象浓度最高的阶段，同时也是最为特殊的——它是幻象维持时间最长、最少被现实替换的阶段。暗恋的结构性条件（对方不知情、信息不完整、关系未经验证）共同构成防腐屏障，使幻象保持在稳定高位，只有极为缓慢的自然下降。暗恋阶段不存在蜜月期反弹，因为它从未经历过现实冲击——没有什么东西可以被重新美化。它是原装的梦，出厂设置，从未开封。

此外，近在咫尺的不可能性（物理空间接近，但关系上无法跨越）所产生的幻象浓度，高于遥远的结构性不可能。前者制造了一种「几乎够得到」的张力——坦塔罗斯悖论——成为幻象持续生长的能量来源。

4.3　热恋阶段

热恋入口是函数中蜜月期反弹最显著的节点之一。进入双向确认的浪漫关系时，「终于可以靠近」的兴奋感激活新一轮幻象建构，但此峰值低于暗恋阶段的稳定水平——因为你已经知道一些真实。Helen Fisher 的神经影像研究显示，热恋期大脑多巴胺奖励回路活跃度与成瘾机制高度重叠，[6] 这解释了为何感受最强烈的阶段，恰恰是幻象快速替换的阶段——感受的真实性与认知的准确性，在此达到最大程度的背离。

4.4　婚姻阶段

婚姻入口同样伴随蜜月期反弹，但峰值低于热恋入口。Murray 的研究支持婚姻中「适度幻象」的功能性——完全去幻象化的长期关系满意度往往更低。[1] Levinas 所描述的「他者的抵抗」在此持续发挥作用：[7] 她永远有你不知道的那一面，这正是她「存在」而不是「被你完全拥有」的证据。

4.5　丧偶阶段与丧偶后期

丧偶造成幻象的短暂骤降，是函数中最低点之一，趋近 ε。丧偶后期则呈现纪念性幻象的二次上扬：死亡切断了现实继续输入的可能性，记忆开始接管，Fading Affect Bias 使负面情绪记忆消退速度快于正面情绪，[8] 逝去伴侣的形象随时间自然神话化。纪念性幻象与蜜月期幻象的根本差异在于性质：前者是期待性的，朝向未来；后者是纪念性的，朝向一个已封存的过去，因此形成稳定高原而不再下降。

第五节

扰动事件的处理

以下事件无法被纳入基准函数，是可在任何关系阶段发生的局部扰动，对幻象程度的影响是非线性的。

⚡
出轨 / 外遇

出轨是幻象的「突变性崩塌」事件，可能在极短时间内造成断崖式下降，逼近甚至短暂突破 ε 下界。复合会触发异常的蜜月期反弹，但峰值通常低于正常热恋期，且下降速度更快。

✈️
异地恋

减少现实输入频率，人为减缓幻象替换速度，使曲线在本应下降的阶段出现局部平稳甚至微幅上升。物理距离介入，使关系的信息结构部分回归「暗恋」状态。久别重逢可触发强烈的蜜月期反弹。

🔄
复合

复合的幻象动态取决于分离原因与时长。多次复合的关系，蜜月期峰值呈逐次递减趋势，反映了「已无更多幻象可供调取」的认知现实。

💔
单方面暗恋的终止（被拒绝 / 自主放弃）

被拒绝是少数能把对方真实意志以不可回避方式注入暗恋者认知系统的事件，造成幻象快速下降。自主放弃则只有极缓慢的自然消退——没有外部事件触发替换，只有时间和其他经历慢慢稀释其浓度。这也解释了为何没有明确结局的单相思，往往需要远比普通失恋更长的时间才能真正放下。

第六节

理论局限与未来研究方向

本框架的核心局限在于量化的不可操作性。「幻象程度」作为心理构念，目前缺乏被广泛接受的测量工具。Murray 与 Holmes 所使用的「理想化评分差值」是目前最接近的间接测量方式，[1] 但它只捕捉了正向理想化的维度。个体系数 k 的量化同样未解决——依附理论提供了潜在的变量框架：焦虑型依附者的幻象程度通常更高、蜜月期峰值更显著；回避型依附者的曲线则更为平坦。[9]

此外，本框架目前仅适用于顺序性浪漫关系。多元恋、无性恋等非主流关系结构，以及不同文化背景下各阶段的定义差异，均可能要求对 X 轴阶段定义进行实质性修订。最后，「恋爱经验难以统计」这一根本性的方法论挑战，使得本框架在可预见的将来仍将停留在思辨性理论层面。但正如普鲁斯特所指出的，爱情本身就是一种认识论的失败——一个要求其研究者也置身其中的命题，或许从一开始就拒绝被客观量化。
